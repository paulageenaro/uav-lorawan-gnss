# Changelog

Todas las versiones y cambios notables de este proyecto se documentarán en este archivo.

## [1.0.0] - 2026-04-28
### Añadido
- Versión inicial del prototipo de nodo LoRaWAN GNSS para TFG.
- Transmisión OTAA en clase A.
- Lectura de tramas NMEA mediante UART usando `TinyGPS++`.
- Construcción de payload binario de 12 bytes.
- Estructura base del repositorio con decodificadores de ChirpStack y documentación.
